# Maestro v2 Reports

This directory contains various reports related to the Maestro v2 workflow system.

## Available Reports

- [Invariants Checklist](./invariants_checklist.md) - Tracks critical system invariants that maintain architectural integrity
- [Inventory Report](../../../../../tools/workflow_v2/inventory.sh) - Run `bash tools/workflow_v2/inventory.sh` to generate an inventory of v2 IR completeness and flag red-alert tokens
